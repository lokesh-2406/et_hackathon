# 🔬 Portfolio Surgeon — AI Money Mentor

🚀 **ET AI Hackathon 2026 | Multi-Agent Financial Intelligence System**

---

## 🧠 What is Portfolio Surgeon?

Portfolio Surgeon is an AI-powered system that analyzes mutual fund portfolios, diagnoses hidden problems, debates investment decisions using multiple AI agents, and generates actionable rebalancing strategies — all within seconds.

Unlike traditional tools, it doesn’t just calculate — it **thinks, debates, and advises**.

---

## ⚡ Key Features

- 📄 Upload CAMS/KFintech PDF
- 📊 Portfolio diagnostics (XIRR, allocation, overlap)
- 🤖 **Debate Club (Unique Feature)**  
  - AI Bull argues to HOLD/BUY  
  - AI Bear argues to SELL  
  - AI Judge gives final verdict  
- 💡 Smart rebalancing strategy
- 📄 Auto-generated investor action memo
- 🚨 Watchlist alerts (SQLite-based tracking)
- 🌐 Interactive Streamlit UI
